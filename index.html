Summary: Latitude vs. X


The purpose of this project was to analyze how weather changes as you get closer to the equator. I pulled data from the OpenWeatherMap API accomplish this analysis.

After accessing the data through an API call to OpenWeatherMap, I assembled a dataset of over 500 random cities around the world. Then, I used Matplotlib to plot various aspects of the weather vs. the latitude, to see what trends I could find. Factors looked at include: temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations associated with this analysis, as well as some explanations and descriptions of any trends or correlations witnessed.
